# Домашние задания с курса «Системный аналитик»

В этой папке собраны учебные работы, выполненные в рамках обучения.  
Тема: Аутентификация и авторизация пользователей в сервисе.


## О работе

Задания выполнялись в рамках учебного курса и иллюстрируют навыки:

- составления технического задания
- моделирования процессов аутентификации и авторизации
- работы с нотациями диаграмм (UML / BPMN / PlantUML)
